🔌 API ENDPOINTS
═══════════════════════════════════════════════════════════════════════════

Base URL: http://127.0.0.1:5000

✓ GET /health
  - System health check
  - Returns: {"status": "healthy", "device": "cuda/cpu"}

✓ GET /api/symptoms
  - Get list of measles symptoms
  - Returns: {"symptoms": {...}, "total_symptoms": 10}

✓ POST /api/predict
  - Make prediction with image & symptoms
  - Body: {image, symptoms, patient_age, patient_gender}
  - Returns: {prediction, confidence, risk_level, recommendation}

✓ GET /api/results
  - Get all predictions (last 100)
  - Returns: {results: [...], total: N}

✓ GET /api/statistics
  - Get dashboard statistics
  - Returns: {total_predictions, measles_cases, average_confidence, ...}

🆘 TROUBLESHOOTING
═══════════════════════════════════════════════════════════════════════════

PROBLEM: Backend won't start
SOLUTION: 
- Ensure Python 3.8+ is installed
- Check if port 5000 is available
- Run: pip install -r requirements.txt

PROBLEM: Frontend won't start
SOLUTION:
- Ensure Node.js is installed
- Delete node_modules and package-lock.json
- Run: npm install
- Run: npm start

PROBLEM: API connection error
SOLUTION:
- Verify backend is running on port 5000
- Check browser console (F12) for errors
- Verify CORS is enabled

PROBLEM: Image upload fails
SOLUTION:
- Check file size (max 5MB)
- Verify file format (JPEG/PNG)
- Check file is not corrupted
