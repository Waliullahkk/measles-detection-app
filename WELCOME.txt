SYMPTOM CHECKLIST (10 ITEMS)
  1. Fever (101-104°F+)
  2. Dry cough
  3. Runny/stuffy nose
  4. Sore throat
  5. Red, watery eyes
  6. Koplik spots
  7. Red rash
  8. Body aches
  9. Extreme fatigue
  10. Loss of appetite

AI-POWERED DETECTION
  • Deep learning model: ResNet18
  • Image analysis: 70% weight
  • Symptom analysis: 30% weight
  • Combined confidence scoring
  • Adjustable threshold: 0.26

RISK ASSESSMENT
  • CRITICAL: High confidence + 5+ symptoms
  • HIGH: High confidence OR 6+ symptoms
  • MODERATE: Mid confidence + 5+ symptoms
  • LOW: Low confidence or non-measles

🚀 HOW TO START

WINDOWS (FASTEST):
─────────────────
1. Open Terminal 1:
   > start_backend.bat
   
2. Open Terminal 2:
   > start_frontend.bat
   
3. Browser: http://localhost:3000

WAIT FOR:
   Backend: "Running on http://127.0.0.1:5000"
   Frontend: "Compiled successfully!"


MAC/LINUX:
──────────
1. Terminal 1:
   $ ./start_backend.sh
   
2. Terminal 2:
   $ ./start_frontend.sh
   
3. Browser: http://localhost:3000

WAIT FOR:
   Backend: "Running on http://127.0.0.1:5000"
   Frontend: "Compiled successfully!"


MANUAL SETUP:
─────────────
Backend (Terminal 1):
  $ cd backend
  $ pip install -r requirements.txt
  $ python app.py

Frontend (Terminal 2):
  $ cd frontend
  $ npm install
  $ npm start
